0. 빈 폴더를 만들어서 vscode로 폴더를 연다
1. ctrl + ` 를 눌러서 터미널을 연다
2. + 오른쪽에 아래 화살표를 눌러서 git bash를 선택한다.
3. 
git clone https://github.com/onetaek/merlot_lab.git .
를 복붙해서 넣는다. -> 그러면 잘받아진다.
4. 공유버튼을 눌러서 pull을 해준다.
5. 그리고 작업을 해준다.
6. 작업이 끝나면 커밋을 해준다.
7. 동기화 버튼을 누른다.
8. 원격에 잘올라갔다!
9. 충돌이 일어나면 햇갈리면 겹치는 부분을 아래로 옮긴다!
